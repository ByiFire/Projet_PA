***Installation/Execution***
-------------------------------------------------------------------------------------------------
**Installation**

Simplement installer les dossier du git, le deziper et ouvrir l'exécutable.

-------------------------------------------------------------------------------------------------
**Formats supportés**

*Uniquement les images en format .bmp sont supportées par l'app.*

-------------------------------------------------------------------------------------------------
**Procédure d’insertion et d’extraction**

Il suffit seulement de cliquer sur le bouton "Charger image" dans l'app, puis chercher votre image au format .bmp dans vos dossiers et l'ouvrir. Si l'image n'est pas dans le format demandé une erreur s'affichera. Pour encoder un message il suffit d'appuyer sur le bouton prévu à cet effet et d'écrire votre message. Et enfin sauvegarder l'image sous un autre nom afin de les différencier, et le tour est joué.

***Installation/Execution***
-------------------------------------------------------------------------------------------------
**Installation** 

Simply install the folder from the repository, unzip it and then open the exe.

-------------------------------------------------------------------------------------------------  
**Supported formats**

*Only images in .bmp format are supported by the app.*

-------------------------------------------------------------------------------------------------
**Insertion and extraction procedure**

Simply click on the ‘Load image’ button in the app, then search for your .bmp image in your folders and open it. If the image is not in the required format, an error message will appear. To encode a message, simply press the button provided for this purpose and write your message. Finally, save the image under a different name to differentiate between them, and you're done.
